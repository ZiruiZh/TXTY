# Privacy Quest - Educational Security Game

An interactive educational game that teaches online privacy and security through realistic scenarios. Navigate through various cybersecurity challenges while being guided (or misled) by TXTY, a devious but cute avatar.

## Features

- **12 Interactive Scenarios**: 8 main scenarios + 4 bonus scenarios covering real-world security situations
- **Desktop UI**: Beautiful faux-desktop interface with windows, taskbar, and notifications
- **TXTY Avatar**: A mischievous guide that provides both helpful and misleading advice
- **Scoring System**: 
  - ✅ Correct answers: 2 points
  - ⚠️ Okay answers: 1 point
  - ❌ Bad answers: 0 points
- **Visual Feedback**: Color-coded responses and detailed explanations
- **Progress Tracking**: Visual progress bar and score breakdown

## Scenarios Covered

### Main Scenarios (1-8)
1. **Library Wi-Fi** - Choosing between secure and open networks
2. **Foreign Login Alert** - Responding to suspicious login notifications
3. **Textbook Hunt** - Identifying malicious file downloads
4. **Panic Email** - Recognizing phishing attempts
5. **Bibliography Trap** - Password security and account creation
6. **Job Offer** - Identifying recruitment scams
7. **Tic Tac Toe Light** - App permission requests
8. **Boredom Quiz** - Social engineering through quizzes

### Bonus Scenarios (9-12)
- **Password Manager Dilemma** - Using password managers
- **Hardware Privacy Check** - Physical security practices
- **Sensitive Group Chat** - End-to-end encryption
- **Digital Footprint** - Metadata and privacy settings

## How to Play

1. Open `index.html` in a modern web browser
2. Read each scenario carefully
3. **Interact with the simulated desktop environment:**
   - Click Wi-Fi icons in the taskbar to select networks
   - Click notification icons to view security alerts
   - Interact with email windows and click links
   - Respond to download dialogs
   - Grant or deny app permissions
   - Fill out forms and make choices
   - Configure social media posts
4. Consider TXTY's advice (but be cautious - he's not always helpful!)
5. Perform actions as you would in real life
6. Learn from the feedback provided after each action
7. Progress through all scenarios
8. Review your final score and grade

**Note:** This is an interactive simulation, not a quiz! You'll perform actual actions like clicking on system tray icons, interacting with popup windows, and making real-world security decisions.

## Scoring

- **A+ (90%+)**: Outstanding! You're a cybersecurity expert!
- **A (80-89%)**: Excellent! Strong security awareness!
- **B (70-79%)**: Good job! You understand the basics well!
- **C (60-69%)**: Not bad, but room for improvement!
